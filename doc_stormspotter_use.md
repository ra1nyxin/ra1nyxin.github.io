# Stormspotter 使用

Stormspotter 用图方式分析 Azure 资源关系和权限路径。

## 常用命令

```bash
stormspotter --help
```

```bash
stormcollector --tenant TENANT_ID
```

```bash
stormspotter-backend
```

```bash
stormspotter --server http://127.0.0.1:9090
```

```bash
az account show
```

小记录：它适合看 Azure 资源关系，采集前先确认租户和订阅边界。
