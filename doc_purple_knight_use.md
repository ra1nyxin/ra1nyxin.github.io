# Purple Knight 使用

Purple Knight 用于 AD 和 Entra ID 安全体检，适合基线评估。

## 常用命令

```bash
PurpleKnight.exe
```

```bash
PurpleKnight.exe /online
```

```bash
PurpleKnight.exe /offline
```

```bash
PurpleKnight.exe /export report.html
```

```bash
PurpleKnight.exe /help
```
