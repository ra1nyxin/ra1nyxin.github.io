# Timeline Explorer 使用

Timeline Explorer 用于查看 Eric Zimmerman 工具导出的 CSV，适合时间线筛选和标注。

## 常用命令

```bash
TimelineExplorer.exe C:\Output\mft.csv
```

```bash
TimelineExplorer.exe C:\Output\prefetch.csv
```

```bash
TimelineExplorer.exe C:\Output\amcache.csv
```

```bash
TimelineExplorer.exe C:\Output\timeline.csv
```

```bash
TimelineExplorer.exe
```

小记录：适合人工浏览大量 CSV，最终结论仍要回到原始证据文件。
