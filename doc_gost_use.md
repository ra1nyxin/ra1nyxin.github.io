# gost 使用

gost 是多协议代理和转发工具，适合临时隧道、代理链和端口转发。

## 常用命令

```bash
gost -L=:1080
```

```bash
gost -L=:8080 -F=socks5://127.0.0.1:1080
```

```bash
gost -L=tcp://:2222/192.168.1.10:22
```

```bash
gost -L=socks5://:1080 -F=http://proxy.local:8080
```

```bash
gost -L=ws://:8080 -F=socks5://127.0.0.1:1080
```

小记录：代理链复杂时要逐段测试，记录每一跳的监听地址和协议。
