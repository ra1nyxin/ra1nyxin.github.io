# PingCastle 使用

PingCastle 用于 Active Directory 风险评分和配置体检。

## 常用命令

```bash
PingCastle.exe --healthcheck
```

```bash
PingCastle.exe --healthcheck --server dc01.domain.local
```

```bash
PingCastle.exe --scanner aclcheck
```

```bash
PingCastle.exe --scanner share
```

```bash
PingCastle.exe --carto
```

它适合管理层报告和整改路线，技术复核仍要回到具体对象和权限。
