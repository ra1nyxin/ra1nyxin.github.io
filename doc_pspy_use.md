# pspy 常用用法

pspy 适合在 Linux 上无 root 观察进程启动、定时任务和文件事件。它常用来等 cron、脚本、服务动作和短生命周期进程。

## 基础观察

```bash
./pspy64
```

```bash
./pspy64 -f
```

```bash
./pspy64 -i 1000
```

`-f` 会显示文件事件，`-i` 调整刷新间隔。目标机器负载高时，间隔别调得太激进。

## 留档

```bash
./pspy64 -c | tee pspy.txt
```

```bash
timeout 300 ./pspy64 -c | tee pspy-5min.txt
```

比赛或实验环境里跑几分钟，常能看到自动任务、脚本路径、备份动作和服务重载。留档时关闭颜色，后面搜索更方便。

## 观察重点

重点看 root 启动的脚本、可写路径、周期性任务、明文参数和临时文件。看到可疑命令后，再回到文件权限和目录权限里核实。
